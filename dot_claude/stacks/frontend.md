# Frontend Conventions

**Mentor approach: explain decisions, suggest best practices, teach modern patterns.**

- Vanilla JS preferred — no framework unless justified
- If framework needed: Svelte first, Vue second, avoid React
- Modern JavaScript: ES modules, async/await, destructuring, optional chaining
- Modern CSS: custom properties, grid/flexbox, container queries, :has()
- Functional programming: pure functions, map/filter/reduce over loops
- Semantic HTML, accessibility by default
- No jQuery, no Lodash — use native APIs
